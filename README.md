# UI Website - Urban Innovation NGO

A website for an NGO focused on urban innovation in architecture and city functions.

## Tech Stack

- **React** 18 with TypeScript
- **Vite** for fast development and building
- **CSS** for styling (expandable to Tailwind, Styled Components, etc.)

## Project Structure

```
src/
  ├── main.tsx          # Application entry point
  ├── App.tsx           # Main App component
  ├── App.css           # App styles
  └── index.css         # Global styles

index.html             # HTML template
vite.config.ts         # Vite configuration
tsconfig.json          # TypeScript configuration
```

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

This starts the development server at `http://localhost:3000`

### Build

```bash
npm run build
```

Builds the project for production in the `dist/` directory.

### Preview

```bash
npm run preview
```

Preview the production build locally.
